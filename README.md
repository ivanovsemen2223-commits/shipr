# ШИПР

Сайт барбершопа в стиле старого плаката.

Демо: https://shipr-nu.vercel.app

## Задача

Сделать одностраничный сайт барбершопа с характером: не модный барбер из торгового центра, а классический мужской салон, где умеют держать опасную бритву. Бренд вымышленный, бриф боевой: запись на стрижку с первого экрана, прайс, команда, галерея, и всё это должно работать без фреймворков и сборки, чтобы жить на любом хостинге.

## Что сделано

Весь проект — три файла: `index.html`, `styles.css`, `app.js`. Локально достаточно открыть `index.html` в браузере.

Эстетика — винтажный плакат: тёплая чёрно-охровая палитра под кофе и патину, конденсированный плакатный шрифт Anton в заголовках, фотографии мастеров за работой. Между секциями крутится классическая красно-сине-белая полоса barber pole, собранная на чистом CSS.

Страница длинная: hero с фото клиента в кресле и заголовками «Бритьё. Стрижка. Уход», форма записи прямо на первом экране, преимущества, рассказ о салоне, табы с типами стрижек от фейда до помпадура, прайс-лист, свёрстанный под кассовый чек, разворот «Стрижка / Бритьё» на два больших фото, команда, цифры, отзыв и галерея.

Из интерактива: заголовки в hero появляются побуквенно, цитата клиента печатается как на машинке, у имён мастеров при наведении срабатывает scramble-эффект, цифры статистики набегают при скролле. Появление секций сделано через IntersectionObserver, всё написано руками без библиотек.

Форма записи — имя, телефон и выбор услуги. Телефон с маской под российский формат и проверкой по мере ввода, после отправки форма показывает подтверждение.

Фон тёпло-чёрный `#0E0D0B`, текст молочный, акцент — золотистая охра `#D4A24A`, для barber pole отдельно заведены красный и синий. Страница полностью адаптивная, на мобильных навигация прячется в бургер-меню. Фото с Unsplash, на случай ошибки загрузки у картинок есть фоллбэк.

## Результат

Готовый сайт без зависимостей: статика, которая деплоится на любой хостинг (сейчас живёт на Vercel) и открывается мгновенно. Дизайн сознательно контрастирует с моим первым кейсом PREMIER — холодный люкс против тёплого ремесла — чтобы в портфолио было видно диапазон. Основа легко переиспользуется под реальный барбершоп, тату-студию или мастерскую.
